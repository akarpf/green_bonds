Python 3.5:
QuantLib, tqdm, lxml, scipy, multiprocessing, math, patsy, dateutil, datetime, pandas, re, numpy, cv2, json

R:
data.table, stargazer, lme4, boot, MASS, zoo, reshape, oaxaca
